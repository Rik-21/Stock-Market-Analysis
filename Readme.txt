Required Modules fot this project:
Numpy: pip install numpy
Scikit-learn: pip install scikit-learn
Keras: pip install keras
plotly: pip install plotly
Yahoo finance: pip install yfinance
Streamlit: pip install streamlit
Datetime: pip install DateTime
Pandas datareader: pip install pandas-datareader
Matplotlib: pip install matplotlib
Pandas: pip install pandas
Time: pip install TIME-python

Write the commands in the VScode terminal to install the respective module.
Also install the jupyter notebook add-on from the store if you are interested in the analysis part.

To check the website run the command:
streamlit run filename.py

Have a look at both the files sma.py and test.py and then choose.
